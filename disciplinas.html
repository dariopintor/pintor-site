Disciplinas.

Cada disciplina tem ementa resumida, bibliografia e acesso aos materiais — slides, listas de exercícios e guias de estudo.

  1. LPG.101

    Lógica de Programação

    Introdução ao pensamento computacional: variáveis, estruturas de decisão, repetição, arrays e funções. Ênfase na construção de algoritmos legíveis e depuráveis por quem ainda está aprendendo a programar.

  2. LPG.102

    Lógica de Programação II

    Aprofundamento: recursão, estruturas de dados básicas (listas, pilhas, filas), complexidade introdutória e modularização. Apresenta boas práticas de refatoração e testes manuais de mesa.

    intermediário estruturas de dados python 60h
    Materiais → Ementa (PDF) →
  3. POO.201

    Programação Orientada a Objetos

    Classes, herança, polimorfismo, encapsulamento e composição. Projeto guiado por responsabilidades, princípios SOLID aplicados na medida certa, e introdução a padrões de projeto clássicos.

    intermediário java SOLID 80h
    Materiais → Ementa (PDF) →
  4. ENG.301

    Engenharia de Software

    Ciclo de vida de software, levantamento de requisitos, modelagem, metodologias ágeis e práticas de qualidade. Projeto em equipe com entregas incrementais e revisão por pares.

    avançado requisitos ágil 80h

    Desenvolvimento Mobile com Swift

    Fundamentos do iOS: UIKit e SwiftUI, arquitetura MVC/MVVM, persistência, integração com APIs e publicação. Foco em construir um aplicativo completo ao longo do semestre.
